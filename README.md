## Installation

This [Quarto](quarto.org) extension can be installed/used using the following command:

- Install
  ```bash
  quarto add emmarshall/quarto-revealjs-emw
  ```
- Use
  ```bash
  quarto use template emmarshall/quarto-revealjs-emw
  ```
